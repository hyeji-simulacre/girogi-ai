# 끼록이 AI

기록과사회 뉴스레터를 학습한 AI 챗봇 - 기록이의 동생, 사이버펑크 거위

## 기능

- 기록학/아카이브 관련 질문 답변
- 뉴스레터 내용 기반 검색 및 요약
- 출처 표시

## 기술 스택

- Streamlit
- Gemini File Search (RAG)
- Python 3.9+

## 실행 방법

```bash
# 환경변수 설정
export GEMINI_API_KEY='your-api-key'

# 실행
streamlit run app.py
```

## 배포

Streamlit Cloud에서 운영 중
